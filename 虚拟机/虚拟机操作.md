0 */5 * * * /home/clear_logs.sh 是一个 Cron 表达式，用于指定在 Linux 系统上定时任务（Cron Job）的执行时间。Cron 表达式用来描述定时任务的执行规则，包括 分、时、日、月 和 星期几 等字段，下面我会详细讲解这个配置的每个部分以及其他常见的 Cron 配置。
Cron 表达式基本结构：
Cron 表达式由五个字段组成，分别表示以下内容：
分   时   日   月   星期
*    *    *   *    *  <命令>
                      分：从 0 到 59 的数字，表示任务在分钟内的执行时间。
                      时：从 0 到 23 的数字，表示任务在小时内的执行时间（24小时制）。
                      日：从 1 到 31 的数字，表示任务在每月的第几天执行。
                      月：从 1 到 12 的数字，表示任务在哪个月执行。
                      星期：从 0 到 7 的数字，表示任务在哪一天执行。0 和 7 都代表星期天，1 代表星期一，依此类推。
                      0 */5 * * * /home/clear_logs.sh 解释：
                      lua
                      复制代码
                      0   */5   *   *   *   /home/clear_logs.sh
                      |    |     |   |   |
                      |    |     |   |   +---- 星期几 (0 - 6) 或 Sun - Sat
                      |    |     |   +-------- 月份 (1 - 12) 或 Jan - Dec
                      |    |     +---------- 日 (1 - 31)
                      |    +------------ 小时 (0 - 23)
                      +------------ 分钟 (0 - 59)
                      这个表达式的含义是：

0：表示在每小时的第 0 分钟 执行，也就是说每次执行会在这个小时的开始时刻运行。
*/5：表示每隔 5 小时 执行一次。这个字段意味着每 5 小时会执行一次，开始时刻是 0 点，即第一次执行会在 00:00，然后是 05:00、10:00、15:00 和 20:00，每 5 小时执行一次。
*：表示每天（每月的每一天）都执行任务。
*：表示每个月都执行任务。
*：表示每个星期的每一天都执行任务（不限制星期几）。

常见的 Cron 配置示例：
每分钟执行一次：
* * * * * /home/clear_logs.sh
          这个表达式表示 每分钟 执行一次 /home/clear_logs.sh 脚本。

每天凌晨 1 点执行：
0 1 * * * /home/clear_logs.sh
这个表达式表示 每天凌晨 1 点 执行脚本。

每个月的 1 号凌晨执行：
0 0 1 * * /home/clear_logs.sh
这个表达式表示 每个月的 1 号 00:00 执行脚本。

每周一凌晨 3 点执行：
0 3 * * 1 /home/clear_logs.sh
这个表达式表示 每周一 03:00 执行脚本。

每 15 分钟执行一次：
*/15 * * * * /home/clear_logs.sh
这个表达式表示 每 15 分钟 执行一次脚本。

每天的 18:30 执行：
30 18 * * * /home/clear_logs.sh
这个表达式表示 每天的 18:30 执行脚本。

每个月的 15 号和 30 号执行：
0 0 15,30 * * /home/clear_logs.sh
这个表达式表示 每个月的 15 号和 30 号 00:00 执行脚本。

每个工作日（周一到周五）每天 8:00 执行：
0 8 * * 1-5 /home/clear_logs.sh
这个表达式表示 每个工作日（周一到周五） 每天 8:00 执行脚本。

每隔两小时执行一次：
0 */2 * * * /home/clear_logs.sh
这个表达式表示 每隔两小时 执行一次脚本，时间点为：00:00，02:00，04:00，06:00 等。

每年 1 月 1 日 00:00 执行一次：
0 0 1 1 * /home/clear_logs.sh
这个表达式表示 每年 1 月 1 日 00:00 执行一次脚本。

Cron 表达式中的特殊字符：
*：代表任何值，表示不限制某一项的值。例如，* 在分钟位置上表示每分钟都执行任务。
*/n：表示每隔 n 个单位执行。例如，*/5 在分钟位置上表示每 5 分钟执行一次。
,：用于分隔多个值。例如，1,5,10 在小时位置上表示在 1 点、5 点和 10 点执行任务。
-：表示范围。例如，1-5 在星期位置上表示从周一到周五执行任务。
/：表示步长，例如 */5 表示每 5 个单位执行一次任务。
L：表示 "last"，可以用在日和星期字段。例如，L 在日字段表示本月最后一天。
W：表示最近的工作日（周一到周五）。例如，15W 表示本月 15 日的最近工作日。
